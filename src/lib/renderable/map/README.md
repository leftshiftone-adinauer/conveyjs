# Map
## JSON structure
```
{
     "markers": [
        {
            "position": {
                "lat": [number]
                "lng": [number]
            },
            "meta": {
                "any": [any]
            },
            "active": [boolean]
        }
     ]  
}
```

#### markers
`markers` array of marker objects

##### position
`position` defines the position of the marker

##### meta
`meta` meta information of the marker

##### active
`active` is marker selected or not selected
